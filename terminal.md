## Mac 隐藏or显示扩展名
-> 点击文件右键 -简介
-> 点击名称与扩展名 即可更改

## Mac 如何看.class文件的内容
-> terminal
-> cd ~/path 
-> javap -c ClassName  *like Java -c HelloWorld

## Mac terminal 编译代码
-> terminal
-> cd to jdk’bin *like cd ~Users/xx/develop/code/
-> javac HelloWorld.java
然后你就会看到一个新的.class的文件啦

## Mac terminal 运行代码
-> terminal
-> cd to jdk’bin *like cd ~Users/xx/develop/code/
-> java HelloWord

## 切换位置和编译和运行小结
-> cd ~/
-> javac xx.java
-> java xx
